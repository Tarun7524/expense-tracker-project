# 💰 Expense Tracker

A simple Expense Tracker web application built using Django.

---

## 🚀 Features

- Add Expenses
- Delete Expenses
- Category Management
- Visual Charts (Chart.js)
- Export to Excel

---

## 🛠 Technologies Used

- Python
- Django
- SQLite
- Chart.js
- OpenPyXL

---

## ⚙️ How to Run

```bash
git clone <your-repo-link>
cd expense-tracker
pip install -r requirements.txt
python manage.py runserver
